Xmlbeans-jxpath

Use JXPath as xpath search engine in XmlBeans

  • Download ZIP

This project is maintained by tunguski

Goal

Use JXPath as XPath engine with XmlBeans objects.

How to use it?

Download

Links on this page

Install

Run Maven to build this project

mvn clean install

Add maven project dependency

    <dependency>
      <groupId>com.tunguski</groupId>
      <artifactId>xmlbeans-jxpath</artifactId>
      <version>1.0</version>
    </dependency>

Search xml element by xpath

import com.tunguski.xmlbeans.XpathSearchUtil;

XpathSearchUtil.selectPath(
  // this is the object from which search begins
  xmlObject,
  // search path
  xpath);
